# Task 3: COVID-19 Dashboard Development (Tableau)

## Objective
To visualize COVID-19 trends using an interactive dashboard.

## Dataset
COVID-19 Open Data (epidemiology.csv)

## Tool Used
- Tableau Public

## Dashboard Visuals
- COVID-19 confirmed cases trend
- Location-wise confirmed cases
- Location-wise death count
- Interactive location filter

## Insights
- Certain locations show rapid growth in confirmed cases
- Death counts vary significantly across regions
- Interactive filters help analyze region-specific trends
