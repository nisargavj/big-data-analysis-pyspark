# Task 2: Predictive Analysis Using COVID-19 Data

## Objective
To perform predictive analysis using machine learning techniques on COVID-19 data.

## Dataset
COVID-19 Open Data (epidemiology.csv)

## Methodology
- Data loading and preprocessing
- Feature selection
- Logistic Regression model training
- Model evaluation

## Insights
- Machine learning can be applied to epidemiological datasets.
- Case trends can be predicted using cumulative and daily metrics.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Google Colab
