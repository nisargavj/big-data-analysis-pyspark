# Task 4: Sentiment Analysis Using NLP

## Objective
To perform sentiment analysis on textual data using NLP techniques.

## Dataset
Twitter US Airline Sentiment Dataset

## Tools & Libraries
- Python
- NLTK
- Scikit-learn

## Methodology
- Text preprocessing
- TF-IDF feature extraction
- Logistic Regression model

## Insights
- Negative sentiments were more common
- NLP preprocessing improved accuracy
- Model effectively classified sentiments
